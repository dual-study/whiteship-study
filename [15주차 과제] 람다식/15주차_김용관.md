- 람다식
    - 메소드를 하나의 식으로 표현한 것
    
    ```java
    //메소드
    int test(int x, int y){
    	return x < y ? x:y;
    }
    
    //람다표현식
    (x,y) -> x < y ? x:y;
    ```
    
    - 자바에서는 클래스의 선언과 동시에 객체를 생성하므로, 단 하나의 객체만을 생성할 수 있는 클래스를 익명 클래스라고 한다. → 자바에서 람다식은 익명 클래스와 같다고 볼 수 있다.
- 람다식 사용법
    
    ```java
    (매개변수목록) -> {함수몸체}
    ```
    
- 함수형 인터페이스
- Variable Capture
- 메소드, 생성자 레퍼런스
