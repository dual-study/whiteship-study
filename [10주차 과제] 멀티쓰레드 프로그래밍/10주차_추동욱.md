# Thread 클래스와 Runnable 인터페이스

## Thread

```java
public class Person extends Thread {

     @Override
     public void run() {
          for (int i=0; i< 10; i++){
               System.out.println("Sub Thread 일 시작: "+ i);
          }
     }
}

public class Main {

    public static void main(String[] args) {

        Person person = new Person();
        person.start();
        
    }
}
```
