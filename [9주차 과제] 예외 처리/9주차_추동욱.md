# 자바에서 예외 처리 방법 (try, catch, throw, throws, finally)

```java
class SomeException extends Throwable{
  public SomeException() {
		super("someException!!")
  }
}

void doSomeThing() throws SomeException{
  throw new SomeException();
}

void main(){
	try{
		doSomeThing()
	}
	catch{
		e.printStackTrace();
	}
	finally{
		System.out.println("finally");
	}
}
```
